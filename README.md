**Problem Statement**
The goal of this project is to analyze NCR ride trip data to uncover ride demand patterns, customer behavior, payment trends, and revenue performance.
The insights from this analysis can help ride-hailing businesses improve operational efficiency, pricing strategies, and customer experience.

**Dataset Overview**
- Ride trip data containing details such as trip status, ride type, payment method, fare amount, trip date, and customer activity
- Data used for structured analysis and business insight generation
- Dataset not included due to file size limitations

**Project Overview**
- Analyze key ride and trip performance metrics
- Understand customer ride behavior and frequency
- Evaluate payment methods and fare distribution
- Identify cancellations and premium ride trends
- Extract actionable insights using SQL

**SQL Techniques Used**
- Data filtering and sorting (WHERE, ORDER BY, LIMIT)
- Aggregations (COUNT, SUM, AVG, MIN, MAX)
- Grouping and conditional analysis (GROUP BY, HAVING, CASE)
- Pattern matching (LIKE)
- Date and time-based analysis
- Data exploration and validation

**Key Business Insights**
- A high percentage of rides are successfully completed, indicating strong service reliability.
- Online payment methods dominate, suggesting a preference for digital transactions.
- Premium rides contribute higher revenue per trip, despite lower ride volume.
- Ride activity peaked in November 2024, indicating seasonal demand trends.
- High-frequency customers show consistent payment behavior, highlighting strong customer loyalty.

**Output & Documentation**
- Each business question is supported by:
- A structured SQL query
- Exported query results (CSV)
- A concise summary of insights

**Tools Used**
- MySQL 8.0
- MySQL Workbench
- Microsoft Excel (for exporting query outputs)

**Conclusion**
This project demonstrates the ability to use SQL for real-world business analysis, transforming raw trip data into meaningful insights.
It highlights skills in data exploration, analytical thinking, and insight communication, making it relevant for Data Analyst and Business Analyst roles.
